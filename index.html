Outdoorsy

Outdoorsy uses Google Maps API, Hiking Project API, and Open Weather API to help users find a hiking trail in any city they'd like. When a user types a city into the search bar, the application saves it in recent searches and generates ten trails in that city. When a user clicks on one of the trails, the details of the trail appear and the map pinpoints the trail's location. Click the image to view the Github repository. View the live site Here
